# Cherry Tree Health Educational Dataset

This repository contains a high-quality, openly licensed dataset and companion teaching guide focused on monitoring and managing cherry tree health. It was developed for use in horticultural education, agricultural research, and AI model training.

## 📂 Contents

- `data/Cherry_Tree_Stress_Education_Dataset.csv` – Structured dataset with checklist rows, stress detection methods, and agronomic notes
- `docs/Teacher_Companion_Guide.pdf` – PDF guide for classroom or workshop use, including teaching prompts and additional context

## 🎓 Purpose

This dataset is designed to support:
- Agricultural science instruction
- AI/ML research in crop health
- Outreach and community farming programs
- Open-access environmental and food system data

## 🗂️ Dataset Fields

Each row includes:
- Topic, Subtopic
- Checklist Item
- Day Range
- Difficulty Level
- Visual Tip
- Educational Notes
- Source
